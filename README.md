# Типа чатик
_Даже немного напоминает нормальный чат_

## Особенности
 - Можно писать сообщения
 - Можно их даже читать
 - Можно отправлять файлы
 - И даже можно их скачивать
 - Файлы конфигурации позволят не вводить данные раз за разом
 - и прочие инновации

 ## Использование
 Запуск сервера:
```sh
./tchat -s
```
 Запуск клиента:
 ```sh
./tchat
```
